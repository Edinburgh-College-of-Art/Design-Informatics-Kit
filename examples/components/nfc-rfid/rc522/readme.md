# MFRC522 RFID Breakout

## Library

Install the following libraries via the Arduino Library Manager

- MFRC522 by Github Community ([GitHub Repo](https://github.com/miguelbalboa/rfid))

## Wiring

### Pin Table

| Nano | MFRC522 |
| ---- | ------- |
| D9   | RST     |
| GND  | GND     |
| N/C  | IRQ     |
| D12  | MISO    |
| D11  | MOSI    |
| D13  | SCK     |
| D10  | SDA     |

**N/C**: Not connected
